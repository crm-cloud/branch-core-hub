# 🏋️ REDESIGN PROMPT: AI Fitness Planner - Diet & Workout Plans

## Current State Analysis (From Screenshots)

**Issues Identified:**
- Generic form layout lacks role-based workflows
- No distinction between Trainer (manual) vs Admin (AI) capabilities
- Missing dietary preference options (veg/non-veg/Indian cuisine)
- No video attachment capability for workout exercises
- No pre-fill from member profile data
- Poor visual hierarchy in plan creation
- Macro calculations not visible
- Member assignment flow needs optimization

---

## REDESIGN OBJECTIVES

### 1. **Role-Based Access Control**
```
TRAINER (Create Manual Plans Only)
├─ Can create custom workout plans
├─ Can create custom diet plans
├─ Can assign to single/multiple members
├─ CANNOT use AI generation
└─ Manual entry only

ADMIN/OWNER (Full AI + Manual)
├─ Can use AI generation (one-click)
├─ Can create manual plans
├─ Can create global templates
├─ Can assign to all members
├─ Can manage plan library
├─ Can view all plans across members
└─ Can edit trainer-created plans
```

### 2. **Member Profile Pre-Fill**
When creating a plan, auto-populate from member database:
```
From Member Profile (Auto-filled):
├─ Age ✓ (from DOB)
├─ Gender ✓
├─ Weight ✓ (current)
├─ Height ✓
├─ BMI ✓ (auto-calculated)
├─ Fitness Level ✓ (from history)
├─ Equipment Available ✓ (home/gym/both)
├─ Current Activity Level ✓
├─ Dietary Restrictions ✓ (saved preferences)
├─ Food Allergies ✓
└─ Injuries/Limitations ✓

User can OVERRIDE any field
```
